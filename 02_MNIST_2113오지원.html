이미지¶

  • 컴퓨터는 이미지 데이터를 픽셀(pixel)들의 모음으로 이해한다. 픽셀은 이미지를 구성하는 최소 단위로 0 ~ 255의 값으로 표현된다.
  • 아래 흑백 이미지는 너비 12개, 높이 16개, 총 12X16=192개의 픽셀로 이루어져있다.
  • 각 픽셀은 0(검정색) ~ 255(흰색)의 값으로 표현되어있는 것을 볼 수 있다. 즉, 컴퓨터는 이미지를 숫자로 받아들이며, 이 숫자를 이용하여 연산을 수행할 수 있다.

  • 컬러 이미지 데이터는 다양한 색을 표현하기 위해 3개의 채널(RGB)을 가진다. 흑백 이미지 데이터는 흑백 표현을 위한 1개의 채널을, 컬러 이미지 데이터는 빨강, 초록, 파랑 표현을 위한 3개의 채널을 지닌다.

1bit 에 두가지 정보 저장
RGB에서 각각은(R,G,B) 8bit이므로 2^24의 색상표현가능

(2^8)^3 = 256^3 =16,777,126가지!

🔗 bit와 색심도

  • 학습 데이터 60,000개, 테스트 데이터 10,000개로 구성

#자릿수를 지정하는 방법 : 정수부 확장 >>%Nd(소숫점 앞으로 N개의 자릿수까지 나타내기 값이 없으면 빈칸), 실수부 확장 >>%0.Nf

문자열 포맷팅(%)¶

대표적인 종류 : %s >>문자열," %d >>정수, %f >>실수

자릿수를 지정하는 방법
정수부 확장 >>%Nd(소숫점 앞으로 N개의 자릿수까지 표현)
실수부 확장 >>%0.Nf(소숫점 뒤로 N개 자릿수까지 표현)

⚠️주의

정수부 확장에서 값이 없으면 빈칸으로 자리가 채워진다

실수부 확장에서 소숫점 2자리까지 표현된수(ex : 2.18)에 대해 %0.5f처럼 지정하면 0으로 채워진다(ex : 2.18000)¶

만약 10자리수에 대해서 %5d처럼 지정하면 아무런 일이 일어나지 않는다

#예제코드
a = "macintosh"
b = 19840124
c = 42.195

print("%s" %a) #문자열
print("%5d" %b) #8자리수에 대해서 정수부 5자리만 지정 >> 아무일도 일어나지 않음
print("%10d" %b) #8자리수에 대해서 정수부 10자리를 지정 >> 앞에 두칸이 빈칸
print("%0.7f" %c) #소숫점 3개까지 나타내어진 수에 대해서 실수부 7자리 지정 >> 뒤에 4칸이 0
print("%0.2f" %c) #소숫점 3개까지 나타내어진 수에 대해서 실수부 2자리 지정 >> 소숫점 두자리까지만
